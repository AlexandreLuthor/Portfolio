Application Java - Magasin de chaussures

Période : Février - Avril 2024

Contexte : Développement d’une application de gestion de ventes pour un magasin de chaussures.

Technologies utilisées

Langages : Java, SQL

Frameworks : JavaFX

Outils : MySQL, NetBeans

Fonctionnalités principales

  • Gestion des stocks (ajout, modification, suppression)
  • Enregistrement des ventes et suivi des transactions
  • Interface utilisateur ergonomique en JavaFX
