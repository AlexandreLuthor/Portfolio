Bibliothèque de Jeux Vidéo

Période : 2024

Contexte : Développement d'une application Java permettant de gérer une bibliothèque de jeux vidéo avec une base de données.

Technologies utilisées

Langages : Java, SQL

Frameworks : JavaFX

Base de données : MySQL

Concepts : MVC, DAO, héritage, navigabilité

Fonctionnalités principales

  • Ajout, suppression et modification de jeux vidéo
  • Gestion des catégories et des éditeurs
  • Affichage et filtrage des jeux par plateforme
  • Import/export des données en JSON, CSV et XML
  • Interface utilisateur intuitive avec JavaFX

Défis techniques et solutions

Gestion des relations dans la base de données : Utilisation de **l'héritage et de la double navigabilité** pour lier les jeux aux éditeurs et catégories.

Optimisation des requêtes : Implémentation du DAO pour structurer les accès à la base de données et éviter les redondances.

Intégration des formats de données : Développement de **fonctions d'exportation en XML, JSON et CSV** pour assurer la compatibilité des données.

Conclusion

Ce projet m’a permis d'approfondir mes compétences en **Java et gestion de bases de données**, tout en appliquant les concepts avancés de programmation orientée objet.
